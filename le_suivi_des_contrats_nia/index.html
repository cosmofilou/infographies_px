Le Suivi Essentiel des Contrats NIA Note d'Information Annuelle

Simplifier la gestion des Notes d'Information Annuelles (NIA) pour Proxiserve et Prox-Hydro.

Découvrez pourquoi et comment nous avons mis en place un suivi centralisé pour tracer les refus des clients à l'option NIA, garantissant notre conformité légale et notre transparence.

Qu'est-ce que la NIA ?

La Note d'Information Annuelle (NIANote d'Information Annuelle) est un document légal obligatoire que nous envoyons à nos clients. Elle détaille la consommation relevée par nos appareils :

  • Répartiteurs de Frais de Chauffage (RFCRépartiteurs de Frais de Chauffage)
  • Compteurs d'Énergie Thermique (CETCompteurs d'Énergie Thermique)
  • Eau Chaude Sanitaire (ECSEau Chaude Sanitaire)

Pourquoi ce suivi est vital ?

Certains clients peuvent légalement refuser cette option facturée. Il est essentiel de tracer ces refus pour :

  • Justifier la non-application en cas de litige.
  • Assurer notre conformité légale.
  • Maintenir la transparence avec nos clients.

Notre Solution : Suivi Centralisé sur Notion

Pour gérer des milliers de contrats, la Direction Technique a mis en place un système robuste sur Notion. Une base de données unique est dupliquée pour chaque agence, contenant toutes les vues nécessaires.

Points Clés :

Sécurité : Chaque agence accède uniquement à ses propres données.
Accès : Connexion via votre adresse e-mail professionnelle.
Responsabilité : Un contact est désigné par agence pour la mise à jour.

Statuts Clairs pour un Suivi Précis

Des statuts spécifiques permettent de suivre l'état de chaque contrat après la facturation initiale.

🟢 NIA Active

Contrat en cours, aucune opposition.

🟠 Refus Unitaire

Refus pour un contrat ou une résidence.

🔴 Refus Total

Refus du client pour tous ses contrats.

Comment Gérer les Refus ?

La procédure de désactivation varie légèrement selon la vue utilisée dans Notion. Sélectionnez l'onglet correspondant à votre cas.

Désactiver un seul contrat

  1. Localisez le contrat et cliquez sur le bouton Désactiver NIA (❌).
  2. Confirmez, puis précisez si la LRARLettre Recommandée avec Accusé de Réception est archivée ou non.
  3. Le statut passe à Refus Unitaire Post-Factu.
  4. Le contrat apparaît dans "Mes Lettres de Refus à Archiver" si nécessaire.

Désactiver tous les contrats d'un client

  1. Recherchez le client pour filtrer la vue.
  2. Sélectionnez tous les contrats affichés via la case à cocher principale.
  3. Dans la barre flottante, changez le "Statut NIA" pour Refus Total Post-Factu.
  4. Archivez la LRARLettre Recommandée avec Accusé de Réception globale sur une seule ligne, puis cochez "LRAR Archivée (☑️)" pour tous les contrats du client.

Désactiver un contrat unitairement

  1. Localisez le contrat et cliquez sur le bouton Désactiver NIA (❌).
  2. Confirmez, puis précisez si la LRARLettre Recommandée avec Accusé de Réception est archivée ou non.
  3. Le statut passe à Refus Unitaire Post-Factu.
  4. Le contrat apparaît dans "Mes Lettres de Refus à Archiver" si nécessaire.

Désactiver tout un groupe

  1. Recherchez le groupe (résidence) pour filtrer la vue.
  2. Sélectionnez tous les contrats du groupe via la case à cocher principale.
  3. Dans la barre flottante, changez le "Statut NIA" pour Refus Unitaire Post-Factu.
  4. Rendez-vous dans "Mes Lettres de Refus à Archiver" pour finaliser le processus d'archivage.

Finaliser l'Archivage des Lettres de Refus

Cette étape est essentielle pour conserver une preuve du refus. La vue "Mes Lettres de Refus à Archiver" liste tous les contrats pour lesquels l'archivage n'a pas encore été effectué.

Procédure d'archivage :

  1. Cliquez directement à l'intérieur du champ de la colonne "Lettre de Refus".
  2. Sélectionnez "Choisir un fichier" pour téléverser le document depuis votre ordinateur.
  3. Une fois le document chargé, cliquez sur le bouton "LRAR Archivée ☑️" à droite de la ligne du contrat.
  4. Le contrat disparaîtra alors de cette vue de suivi.

Le Suivi de la Relève : Votre Guide Simplifié

Comprenez comment les relevés de compteurs sont traités, de la planification à la mise à disposition de vos documents.

Cette infographie vous guide à travers les étapes clés du processus de suivi de la relève. Découvrez le parcours complet d'un index, depuis le terrain jusqu'à votre espace client.

Flux Principal : Le Suivi de la Relève

  1. 1. Planification des Relevés

    Organisation des équipes : Les relevés sont planifiés par semaine et par équipe pour optimiser les tournées et garantir une couverture efficace.

    Création des supports numériques : Des supports (applications sur smartphones ou tablettes) sont préparés pour les releveurs, contenant toutes les informations nécessaires pour leur mission.

  2. 2. Collecte des Données

    La collecte des index s'effectue de deux manières complémentaires :

    • Sur le terrain (Mobilité) : Nos releveurs équipés de leurs appareils mobiles saisissent directement les index.
    • À distance (Télérelevé GPRS) : Pour les compteurs équipés, les index sont collectés automatiquement.

  3. 3. Intégration des Index

    Les données sont transférées dans le système Licorne. Fréquence d'intégration :

    • Hebdomadaire : Chaque vendredi, les données des télérelevés sont automatiquement intégrées.
    • À date imposée : Dans certains cas spécifiques, une date précise est définie pour l'intégration.

  4. 4. Validation Automatique

    Licorne vérifie automatiquement chaque relevé. Critères clés vérifiés :

    Contrat valide : L'affaire est active.
    Index cohérent : Supérieur au précédent.
    Pas d'anomalie : Absence de codes bloquants.
    Compteur non déposé : Le compteur général est en service.
    Consommation normale : Sous les seuils d'alerte.
    Validation débloquée : Pas de blocage manuel activé.

  5. 5. Dépôt des Documents (Éditiques)

    • Sur Accès Pro : Une fois les relevés validés, les documents officiels sont automatiquement déposés sur votre espace client.
    • Fichiers EDI : Si configuré, un fichier EDI est généré et déposé sur Accès Pro.

    Adieu les e-mails !

    Pour une meilleure sécurité et traçabilité, l'envoi des éditiques par e-mail est remplacé par le dépôt direct sur Accès Pro.

Gestion des Anomalies

Que se passe-t-il si la validation automatique échoue ou en cas de traitement manuel ?

Validation KO (Critères Non OK)

Si un critère de validation n'est pas respecté, le relevé est marqué "Validation Auto KO". Il ne sera pas validé et nécessitera une révision et une intervention manuelle par nos équipes pour corriger l'anomalie.

Traitement manuel

Même si une demande de travail (DT) est traitée manuellement en dehors du processus automatique, soyez assuré que l'éditique final sera quand même déposé sur Accès Pro une fois le cas résolu.
